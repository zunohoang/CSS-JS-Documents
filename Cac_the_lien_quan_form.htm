Các Thuộc Tính CSS Liên Quan Đến Thẻ <input>, <button>, và Các Thẻ Form

CSS cung cấp một số thuộc tính để tuỳ chỉnh giao diện và hành vi của các thẻ <input>, <button>, và các thẻ liên quan trong form. Dưới đây là một số thuộc tính cơ bản mà bạn có thể sử dụng để tùy chỉnh các thẻ này.

1. Thuộc tính cho thẻ <input>

width và height

Thuộc tính này dùng để thay đổi kích thước của thẻ <input>.

Ví dụ:

input {
  width: 200px;
  height: 40px;
}

Các đặc điểm:

  • Bạn có thể thay đổi chiều rộng và chiều cao của thẻ <input> để phù hợp với giao diện của trang web.

padding

Thuộc tính padding giúp tạo không gian bên trong thẻ <input>, giữa nội dung và viền.

Ví dụ:

input {
  padding: 10px;
}

Các đặc điểm:

  • Tạo không gian trong thẻ nhập liệu giúp người dùng dễ dàng nhập văn bản hơn.

border

Thuộc tính border thay đổi kiểu, độ rộng và màu sắc của viền xung quanh thẻ <input>.

Ví dụ:

input {
  border: 2px solid #3498db;
}

Các đặc điểm:

  • Điều chỉnh viền của thẻ <input>, giúp tăng tính thẩm mỹ hoặc làm nổi bật các ô nhập liệu.

border-radius

Thuộc tính border-radius được dùng để tạo bo góc cho thẻ <input>, giúp các thẻ input trông mềm mại hơn.

Ví dụ:

input {
  border-radius: 10px;
}

Các đặc điểm:

  • Giúp các thẻ input có góc bo tròn, tạo cảm giác thân thiện và dễ sử dụng.

background-color

Thuộc tính này điều chỉnh màu nền của thẻ <input>.

Ví dụ:

input {
  background-color: #f4f4f4;
}

Các đặc điểm:

  • Cải thiện giao diện của thẻ <input> bằng cách thay đổi màu nền.

font-size và font-family

Thuộc tính font-size và font-family giúp bạn thay đổi kiểu chữ và kích thước chữ trong thẻ <input>.

Ví dụ:

input {
  font-size: 16px;
  font-family: Arial, sans-serif;
}

Các đặc điểm:

  • Điều chỉnh kiểu chữ và kích thước chữ giúp thẻ <input> phù hợp hơn với giao diện tổng thể của trang web.

outline

Thuộc tính outline dùng để điều chỉnh viền ngoài của thẻ <input>. Nó giống như border, nhưng không chiếm không gian.

Ví dụ:

input:focus {
  outline: 3px solid #3498db;
}

Các đặc điểm:

  • Thường được sử dụng để tạo hiệu ứng khi người dùng chọn thẻ <input>.

box-shadow

Thuộc tính box-shadow giúp tạo hiệu ứng bóng đổ cho thẻ <input>.

Ví dụ:

input {
  box-shadow: 0px 4px 6px rgba(0, 0, 0, 0.1);
}

Các đặc điểm:

  • Tạo bóng cho thẻ <input> giúp tạo hiệu ứng nổi bật cho các ô nhập liệu.

:focus

Pseudo-class :focus áp dụng cho thẻ <input> khi người dùng kích hoạt nó (ví dụ: khi nhấp vào ô nhập liệu).

Ví dụ:

input:focus {
  border-color: #3498db;
}

Các đặc điểm:

  • Thường dùng để thay đổi kiểu dáng của thẻ input khi người dùng tương tác với nó.

2. Thuộc tính cho thẻ <button>

padding

Thuộc tính padding giúp tạo không gian bên trong thẻ <button>, giữa nội dung và viền.

Ví dụ:

button {
  padding: 10px 20px;
}

Các đặc điểm:

  • Tạo không gian giúp nút bấm trở nên dễ nhấn hơn và nhìn đẹp mắt hơn.

border

Thuộc tính border thay đổi kiểu, độ rộng và màu sắc của viền nút <button>.

Ví dụ:

button {
  border: 2px solid #3498db;
}

Các đặc điểm:

  • Điều chỉnh viền của nút <button>, giúp nút nổi bật hơn hoặc hòa nhập với thiết kế của trang.

background-color

Thuộc tính này điều chỉnh màu nền của thẻ <button>.

Ví dụ:

button {
  background-color: #3498db;
}

Các đặc điểm:

  • Cung cấp màu nền cho nút để phù hợp với giao diện của trang web.

color

Thuộc tính color dùng để thay đổi màu chữ trong thẻ <button>.

Ví dụ:

button {
  color: white;
}

Các đặc điểm:

  • Điều chỉnh màu sắc của văn bản trong nút.

font-size và font-family

Thuộc tính font-size và font-family giúp bạn thay đổi kiểu chữ và kích thước chữ trong thẻ <button>.

Ví dụ:

button {
  font-size: 18px;
  font-family: Arial, sans-serif;
}

Các đặc điểm:

  • Cải thiện giao diện của nút bằng cách thay đổi kiểu và kích thước chữ.

border-radius

Thuộc tính border-radius giúp tạo bo góc cho thẻ <button>, làm nút trở nên mềm mại và dễ nhìn hơn.

Ví dụ:

button {
  border-radius: 5px;
}

Các đặc điểm:

  • Tạo hiệu ứng bo góc cho nút, mang lại cảm giác thân thiện.

box-shadow

Thuộc tính box-shadow giúp tạo bóng đổ cho thẻ <button>, tạo hiệu ứng chiều sâu.

Ví dụ:

button {
  box-shadow: 0px 4px 6px rgba(0, 0, 0, 0.1);
}

Các đặc điểm:

  • Tạo hiệu ứng bóng cho nút, làm cho nó nổi bật hơn khi người dùng nhìn thấy.

:hover

Pseudo-class :hover được sử dụng để áp dụng các thay đổi khi người dùng di chuột lên nút.

Ví dụ:

button:hover {
  background-color: #2980b9;
}

Các đặc điểm:

  • Thường được dùng để thay đổi màu sắc hoặc hiệu ứng khi người dùng di chuột qua nút.

:focus

Pseudo-class :focus dùng khi nút <button> được chọn.

Ví dụ:

button:focus {
  outline: none;
}

Các đặc điểm:

  • Thường được sử dụng để thay đổi kiểu dáng của nút khi người dùng kích hoạt nó.

3. Thuộc tính cho thẻ <select> và <textarea>

width và height

Thuộc tính này thay đổi kích thước của thẻ <select> và <textarea>.

Ví dụ:

select, textarea {
  width: 300px;
  height: 100px;
}

Các đặc điểm:

  • Giúp điều chỉnh kích thước của các thẻ <select> và <textarea>.

border

Thuộc tính border dùng để thay đổi kiểu, độ rộng và màu sắc của viền.

Ví dụ:

select, textarea {
  border: 1px solid #3498db;
}

Các đặc điểm:

  • Điều chỉnh viền của thẻ <select> và <textarea>.

padding

Thuộc tính này tạo không gian bên trong thẻ <select> và <textarea>, giúp nội dung dễ nhìn và dễ nhập hơn.

Ví dụ:

select, textarea {
  padding: 10px;
}

Các đặc điểm:

  • Tạo không gian cho văn bản bên trong thẻ.

Kết Luận

CSS cung cấp rất nhiều thuộc tính hữu ích để tùy chỉnh giao diện của các thẻ form như <input>, <button>, <select>, và <textarea>. Việc hiểu rõ cách sử dụng các thuộc tính này giúp bạn tạo ra các giao diện người dùng dễ sử dụng và thẩm mỹ hơn.
