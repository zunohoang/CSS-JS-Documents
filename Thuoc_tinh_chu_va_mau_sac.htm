Các Thuộc Tính CSS Liên Quan Đến Chữ và Màu Sắc

CSS cung cấp rất nhiều thuộc tính để điều chỉnh các yếu tố văn bản và màu sắc của văn bản. Dưới đây là các thuộc tính cơ bản mà bạn cần biết.

1. color

Thuộc tính color được sử dụng để thay đổi màu sắc của chữ.

Cách sử dụng:

p {
  color: #ff5733; /* Mã màu hex */
}

Hoặc có thể sử dụng tên màu:

p {
  color: red;
}

Các giá trị có thể sử dụng:

  • Mã màu hex (ví dụ: #ff5733)
  • Tên màu (ví dụ: red, blue, green)
  • Giá trị RGB (ví dụ: rgb(255, 0, 0))
  • Giá trị RGBA (với độ trong suốt: rgba(255, 0, 0, 0.5))
  • Giá trị HSL (ví dụ: hsl(0, 100%, 50%))

2. background-color

Thuộc tính background-color thay đổi màu nền của phần tử chứa văn bản.

Cách sử dụng:

p {
  background-color: yellow;
}

3. font-family

Thuộc tính font-family xác định kiểu chữ sẽ được sử dụng cho văn bản trong phần tử.

Cách sử dụng:

p {
  font-family: 'Arial', sans-serif;
}

Bạn có thể chỉ định nhiều kiểu chữ để hệ thống chọn kiểu chữ đầu tiên có sẵn.

4. font-size

Thuộc tính font-size điều chỉnh kích thước chữ.

Cách sử dụng:

p {
  font-size: 18px;
}

Các đơn vị có thể sử dụng:

  • px: Pixels (đơn vị cố định)
  • em, rem: Đơn vị tương đối (tùy thuộc vào kích thước font gốc)
  • %: Đơn vị phần trăm, dùng để thay đổi kích thước văn bản tương ứng với kích thước font gốc.

5. font-weight

Thuộc tính font-weight dùng để điều chỉnh độ đậm của chữ.

Cách sử dụng:

p {
  font-weight: bold;
}

Các giá trị có thể sử dụng:

  • normal: Độ đậm mặc định.
  • bold: Đậm.
  • lighter: Nhạt hơn.
  • Số (100, 200, ..., 900): Để xác định mức độ đậm.

6. font-style

Thuộc tính font-style được sử dụng để thay đổi kiểu chữ, ví dụ như chữ nghiêng.

Cách sử dụng:

p {
  font-style: italic;
}

Các giá trị có thể sử dụng:

  • normal: Kiểu chữ bình thường.
  • italic: Kiểu chữ nghiêng.
  • oblique: Kiểu chữ nghiêng nhẹ.

7. line-height

Thuộc tính line-height điều chỉnh chiều cao dòng, giúp tạo khoảng cách giữa các dòng chữ.

Cách sử dụng:

p {
  line-height: 1.6;
}

Giá trị có thể là một tỷ lệ (ví dụ: 1.5, 2) hoặc đơn vị độ dài (ví dụ: 20px).

8. letter-spacing

Thuộc tính letter-spacing điều chỉnh khoảng cách giữa các ký tự trong văn bản.

Cách sử dụng:

p {
  letter-spacing: 2px;
}
  • letter-spacing có thể nhận giá trị như px, em, rem, ...

9. word-spacing

Thuộc tính word-spacing điều chỉnh khoảng cách giữa các từ trong văn bản.

Cách sử dụng:

p {
  word-spacing: 5px;
}

10. text-align

Thuộc tính text-align xác định cách căn chỉnh văn bản trong phần tử.

Cách sử dụng:

p {
  text-align: center;
}

Các giá trị có thể sử dụng:

  • left: Căn trái.
  • right: Căn phải.
  • center: Căn giữa.
  • justify: Căn đều hai bên.

11. text-transform

Thuộc tính text-transform thay đổi cách hiển thị chữ (chuyển tất cả sang hoa, thường, hay viết hoa chữ cái đầu tiên của mỗi từ).

Cách sử dụng:

p {
  text-transform: uppercase;
}

Các giá trị có thể sử dụng:

  • uppercase: Chuyển tất cả chữ thành chữ hoa.
  • lowercase: Chuyển tất cả chữ thành chữ thường.
  • capitalize: Chỉ viết hoa chữ cái đầu tiên của mỗi từ.

12. text-decoration

Thuộc tính text-decoration dùng để thêm hiệu ứng trang trí cho văn bản (như gạch chân, gạch ngang).

Cách sử dụng:

a {
  text-decoration: underline;
}

Các giá trị có thể sử dụng:

  • none: Không có trang trí.
  • underline: Gạch chân.
  • line-through: Gạch ngang.
  • overline: Gạch trên.

13. text-shadow

Thuộc tính text-shadow tạo bóng cho văn bản.

Cách sử dụng:

h1 {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}

Các giá trị:

  • 2px 2px: X và Y của bóng.
  • 4px: Độ mờ của bóng.
  • rgba(0, 0, 0, 0.5): Màu và độ trong suốt của bóng.

14. white-space

Thuộc tính white-space điều khiển cách các ký tự trắng (như khoảng trắng, tab, dòng mới) được xử lý trong văn bản.

Cách sử dụng:

p {
  white-space: nowrap;
}

Các giá trị có thể sử dụng:

  • normal: Các ký tự trắng sẽ được xử lý bình thường.
  • nowrap: Ngăn cách các dòng mới, nội dung sẽ được hiển thị trong một dòng.
  • pre: Giữ nguyên khoảng trắng và dòng mới.

15. direction

Thuộc tính direction điều chỉnh hướng văn bản, dùng trong các ngôn ngữ viết từ phải sang trái như tiếng Ả Rập hoặc tiếng Do Thái.

Cách sử dụng:

p {
  direction: rtl;
}

Các giá trị:

  • ltr: Từ trái sang phải (mặc định).
  • rtl: Từ phải sang trái.

Kết Luận

Các thuộc tính CSS liên quan đến chữ và màu sắc giúp bạn kiểm soát các yếu tố trực quan quan trọng trên trang web, bao gồm kiểu chữ, màu sắc, độ đậm, căn chỉnh và nhiều hiệu ứng khác. Việc sử dụng đúng các thuộc tính này sẽ giúp bạn tạo ra một giao diện đẹp mắt và dễ đọc hơn.
