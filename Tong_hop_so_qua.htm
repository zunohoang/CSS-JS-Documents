Tổng hợp kiến thức CSS

1. Giới thiệu về CSS

  • CSS (Cascading Style Sheets) là ngôn ngữ định kiểu được sử dụng để định dạng nội dung HTML.
  • CSS cho phép thay đổi giao diện và bố cục của website.

1.1 Cách chèn CSS vào HTML

  1. Inline CSS: Viết trực tiếp vào thuộc tính style của phần tử. 
    <p style="color: red;">Văn bản màu đỏ</p>
  2. Internal CSS: Viết trong thẻ <style> trong file HTML. 
    <style>
    p {
        color: blue;
    }
</style>
  3. External CSS: Liên kết file .css bên ngoài bằng thẻ <link>. 
    <link rel="stylesheet" href="styles.css">

2. Các khái niệm cơ bản

2.1 Selectors (Bộ chọn)

  • Selector dùng để chọn và áp dụng style cho các phần tử HTML.
    • Bộ chọn cơ bản:
      • *: Chọn tất cả phần tử.
      • element: Chọn theo tên thẻ, ví dụ p, div.
      • .class: Chọn theo class, ví dụ .button.
      • #id: Chọn theo ID, ví dụ #header.
    • Bộ chọn kết hợp:
      • div p: Chọn tất cả <p> bên trong <div>.
      • h1, h2: Chọn cả <h1> và <h2>.
      • div > p: Chọn tất cả <p> là con trực tiếp của <div>.
      • div + p: Chọn phần tử <p> ngay sau <div>.

2.2 Thuộc tính CSS cơ bản

  • Màu sắc: color, background-color 
    color: red;
background-color: #f0f0f0;
  • Kích thước: width, height, max-width, max-height 
    width: 100px;
height: auto;
  • Bố cục: margin, padding, border 
    margin: 10px;
padding: 20px;
border: 1px solid black;

2.3 Box Model

  • Gồm 4 phần: content, padding, border, margin.

3. Các khái niệm nâng cao

3.1 Flexbox

  • Dùng để căn chỉnh và bố trí phần tử. 
    display: flex;
justify-content: center; /* Căn giữa theo chiều ngang */
align-items: center; /* Căn giữa theo chiều dọc */

3.2 Grid Layout

  • Cách tạo lưới trên giao diện. 
    display: grid;
grid-template-columns: repeat(3, 1fr);
grid-gap: 20px;

3.3 Responsive Design

  • Đảm bảo giao diện hiển thị tốt trên mọi thiết bị. 
    @media (max-width: 768px) {
    body {
        background-color: lightblue;
    }
}

3.4 Animation

  • Thêm hiệu ứng chuyển động. 
    @keyframes fadeIn {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}
div {
    animation: fadeIn 2s ease-in-out;
}
